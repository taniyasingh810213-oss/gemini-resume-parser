# 🧠 Gemini AI Resume Parser UI

A beautiful, custom-built desktop application that allows users to upload resumes (PDFs or text files) and instantly extract structured information like name, email, skills, and work experience using the Gemini AI API.

## ✨ Features
* **Modern GUI:** Built with a clean, dark-themed Tkinter interface.
* **File Support:** Easy "Choose File" button supporting both PDF and standard text formats.
* **Smart Extraction:** Powered by `gemini-3-flash` to accurately parse unstructured resume text.
* **Secure:** Uses environment variables to keep API keys completely safe.
